Useful information
V
  • You can submit your game here https://developer.crazygames.com/submit
  • There is no need to upload this template file. You only need to upload the "Build" folder, and "StreamingAssets" if you have one.
  • Have you optimized your build size as much as possible? A smaller build results in a higher load rate, which in turn increases your income. The WebGL optimizer that comes with the SDK may help you do this, and you can access it in Unity under this menu option CrazyGames > WebGL Optimizer. Alternatively, you can check these export tips.
  • Outside of CrazyGames the SDK will run in demo mode, you'll see demo ads and dummy data. In order to run your game in Crazygames-like conditions, you can submit your game or use our QA Tool to test it.
  • If you wish to upload the game on your domain, don't forget to whitelist it in the CrazySDK settings, otherwise, the sitelock will block it and the game won't start. CrazySDK will not be active on your domain, and calls to it will do nothing.
