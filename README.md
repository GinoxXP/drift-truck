# Drift Truck

![Gameplay](/.promo/banner.png)

A game about delivery service

![Gameplay](/.promo/gameplay.gif)

Drift truck is a racing casual game about delivering cargo from point
A to point B. The player needs to be able to control the speed of the car,
but can turn left and right. When turning, the car begins to drift
and slow down. Otherwise, the car accelerates.
